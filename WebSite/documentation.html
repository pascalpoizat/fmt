Installation for use independently from the Eclipse IDE

VerChor can be used as a command-line tool. In addition to the VerChor distribution (see here) you need to install:

  • a Java RE (needed to transform BPMN files into the CIF Format)
    download Java RE
    Note: VerChor has been tested with Java RE 1.7.
  • a Python distribution and the PyXB library (needed to read CIF files and generate formal description files)
    download Python 2.7.x download PyXB 1.1.1
    Warning: Python 2.7.x and PyXB 1.1.1 should be used.
    Newer versions do not work correctly with VerChor.
  • the CADP toolbox (needed to perform formal analyses on the formal description files)
    download CADP
Then please take a look at the readme.txt in the VerChor distribution.

Installation for use in integration with the Eclipse IDE

VerChor takes its full power when integrated with the Eclipse IDE and its BPMN plugin. In addition to the independent installation above, you need to install:

  • the Eclipse IDE and the BPMN modeler plugin
    download Eclipse download BPMN modeler plugin
    Note: VerChor has been tested with Eclipse Indigo (3.7), Eclipse Juno (4.2), and Eclipse Kepler (4.3).
    We advocate for the use of the later due to a better integration of the BPMN modeler plugin (tested with Eclipse 4.3.1 and BPMN modeler plugin 1.0).
Then please take a look at the readme.txt in the VerChor distribution.
